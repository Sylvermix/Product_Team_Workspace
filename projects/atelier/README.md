## Files

- `context.md` — vision, audience, aesthetic, stack, constraints (read this first)
- `backlog.yaml` — prioritized stories + discovery spikes
- `roadmap.yaml` — current sprint + milestones
- `changelog.md` — append-only shipped history
- `design_system/tokens.yaml` — atelier-specific design tokens (editorial × minimal)
- `design_system/components.yaml` — component registry (populated as Design agent creates components)
- `specs/` — feature specs produced by Design for Tech
- `adr/` — architecture decision records by Tech
